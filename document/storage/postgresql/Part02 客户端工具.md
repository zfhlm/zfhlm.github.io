
# postgresql 客户端工具

### navicat for postgresql

    下载并安装客户端，地址：http://www.navicat.com.cn/download/navicat-for-postgresql

    打开 navicat 新建连接，填写以下信息：

        连接名：192.168.140.133

        主机：192.168.140.133

        端口：5432

        初始数据库：postgres

        用户名：postgresql

        密码：123456

    点击【测试连接】，成功后点击【确定】，至此完成客户端配置
